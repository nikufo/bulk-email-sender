# Email Sender Tool

A powerful desktop application for sending bulk emails with ease.

## Features

*   **Multiple SMTP Support**: Manage and rotate between multiple email accounts.
*   **Proxy Support**: Use SOCKS5 proxies for secure sending.
*   **Bulk Sending**: Send emails to large lists with customizable delays and batch limits.
*   **Attachment Support**: Easily attach files to your emails.
*   **HTML Templates**: Send beautiful HTML emails.
*   **Real-time Logging**: Track the status of every email sent.

## Installation

1.  Download the `EmailSenderTool.exe` file.
2.  Run the executable.
3.  Follow the on-screen setup wizard to configure your email accounts.

## Usage

1.  **Add Accounts**: Go to the Settings tab to add your SMTP accounts (Gmail, Outlook, etc.).
2.  **Import Contacts**: Load your email list from a CSV or Excel file.
3.  **Compose**: Write your subject and message (or load an HTML template).
4.  **Send**: Click "Start Sending" and watch the progress.
